# zimmyb-site

===

This theme is built on Wordpress Underscores (but does not fully follow it's workflow) : http://underscores.me

Readme Table of Contents:
1. Setup
2. WP Plugins

Setup
---------------
```
> git clone https://github.com/camachowebworld/zimmyb-site.git
> cd zimmyb-site
> npm install
> grunt
```
